# Install

Reference [here](https://github.com/passcod/cargo-watch).
```
# Install.
$ cargo install cargo-watch

# Upgrade.
$ cargo install --force cargo-watch
````

## Rustfmt

```bash
$ cargo +nightly fmt
```

## Start

```bash
$ cargo watch -x 'run'
```

## CLI

```bash
$ cargo run -- --help
$ cargo run -- run --address 127.0.0.1:4567
```
